# React Register Form (React Hook Form + Yup)

## Description
A simple registration form built with **React Hook Form** and **Yup** validation.  
The form validates user input and displays errors inline.

## Features
- Full Name (required, min 3 characters)
- Email (required, valid format)
- Password (required, min 8 characters, at least 1 number)
- Confirm Password (must match password)
- Terms & Conditions checkbox (required)
- Success message on valid submit
- Form data logged to console

## Technologies Used
- React
- react-hook-form
- yup
- @hookform/resolvers
